下面看一下React Native iOS 默认代码

RCTRootView *rootView 这个我把它理解为一个容器控件，容纳React Native代码。（至此解决了另一个困惑，在一个已经成熟的项目中使用React Native技术，直接插入这个模块即可）

iOS 读取的是 index.ios.js 这个文件，同理android 读取的是index.android.js。
这个文件是React Native 的入口文件。 相当于main 亦或者是iOS的appDelegate 文件。
