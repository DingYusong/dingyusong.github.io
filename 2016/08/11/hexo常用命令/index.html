Hexo 基本资料

1.官网

最好的学习的地方https://hexo.io/

从一个全新的MAC到能使用hexo写博客

1.想直接跑起来本地服务发现 hexo 没有安装

hexo -s   
    -bash: hexo: command not found

so 安装hexo

2.安装hexo时发现npm没有安装

npm install hexo-cli -g
-bash: npm: command not found

so 安装npm

3.同时想写一份笔记记录，于是想安装一个MACDown软件 发现mac包管理器Brew没有安装

brew cask install macdown
-bash: brew: command not found

so 安装brew

4.安装brew

官方地址：
http://brew.sh/index_zh-cn.html

/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

安装成功

==> Installation successful!
==> Next steps
Run `brew help` to get started

homeBrew的使用教程
http://www.cnblogs.com/TankXiao/p/3247113.html
brew search 是查找
brew search /wge*/

5.安装macDown

brew cask install macdown

等了好久发现不行，于是连上VPN 成功！科学上网总能解决很多问题，你懂得~

6.继续安装npm

http://www.liaoxuefeng.com/wiki/001434446689867b27157e896e74d51a89c25cc8b43bdb3000/00143450141843488beddae2a1044cab5acb5125baf0882000

npm已经在Node.js安装的时候顺带装好了

说明需要先安装node

7.安装node

去node的官网下载安装包直接安装，目前是V6.4.0。当然也可以使用brew方式安装

<https://nodejs.org/en/>

安装后查看一下版本号

localhost:next Ding$ npm -v
3.10.3
localhost:next Ding$ node -v
v6.4.0

8.现在安装hexo

npm install hexo-cli -g

正常情况应该安装成功的，现在报错

npm ERR! Error: EACCES: permission denied, access '/usr/local/lib/node_modules'
npm ERR!     at Error (native)
npm ERR!  { Error: EACCES: permission denied, access '/usr/local/lib/node_modules'
npm ERR!     at Error (native)
npm ERR!   errno: -13,
npm ERR!   code: 'EACCES',
npm ERR!   syscall: 'access',
npm ERR!   path: '/usr/local/lib/node_modules' }
npm ERR! 
npm ERR! Please try running this command again as root/Administrator.

查找解决方法

http://stackoverflow.com/questions/16151018/npm-throws-error-without-sudo

我怀疑是直接安装pkg包导致的权限问题。so用brew再次安装下试一试。

brew install node

发现不行，那就直接修改权限

sudo chown -R $(whoami) ~/.npm

发现也不行

所以在前面加sudo 试一试
sudo npm install hexo-cli -g

安装成功！

9.使用hexo

hexo -s

失败

ERROR Local hexo not found in ~/Documents/GitRepe/blogs
ERROR Try running: 'npm install hexo --save'



npm install hexo --save

成功

10.安装hexo-server

npm install hexo-server --save
